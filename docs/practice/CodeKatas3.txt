- Schreibe eine Klasse Printer, die über mehrere überladene print-Methoden verfügt. Eine Methode soll eine Zeichenkette ausdrucken, eine andere Methode soll eine Ganzzahl ausdrucken, und eine dritte Methode soll ein Array von Ganzzahlen ausdrucken. Teste alle drei Methoden in der main-Methode, um ihre Funktionalität zu demonstrieren.

- Erstelle eine Methode calculateAverage, die eine variable Anzahl von double-Werten (double... numbers) akzeptiert. Die Methode soll den Durchschnitt der übergebenen Zahlen berechnen und zurückgeben. Überprüfe die Funktion in main mit verschiedenen Zahlenlisten.
- Schreibe eine Klasse Printer, die zwei überladene Methoden print enthält. Die erste Methode akzeptiert Varargs von String-Objekten und die zweite Methode Varargs von int. Jede Methode soll alle übergebenen Werte in einer Zeile ausgeben. Teste beide Methoden in main mit unterschiedlichen Eingaben.
